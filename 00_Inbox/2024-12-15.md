	#独立开发 #营销
	
	如果没有营销预算，试试这样去推广你的创业项目
	（建议收藏需要的时候多来看看）:
	
	1. 体验式二维码营销
	    
	    → 用吸引人的内容抓住用户眼球
	    
	    → 引导用户扫描二维码
	    
	    → 将用户引导至落地页
	    
	    → 让用户进入转化漏斗
	    
	
	2. 在亚马逊 KDP 平台发布电子书引流
	    
	    → 创建一本49页的电子书
	    
	    → 针对目标客户群(ICP)的真实痛点
	    
	    → 在书籍封面添加网站链接
	    
	    → 注册 Amazon KDP 账号
	    
	    → 创建作者主页以提升曝光
	
	    
	3. 通过 Presspulse .ai 寻找记者合作
	    
	    → 完善个人资料
	    
	    → 添加背景和专业领域
	    
	    → 与优质记者匹配
	    
	    → 回答他们的问题
	    
	    → 获得免费PR机会
	    
	
	4. 通过 Show HN 获得数百次曝光
	    
	    → 访问 Y Combinator 的 Hacker News 版块
	    
	    → 点击"Show"
	    
	    → 用一句话描述你的业务
	    
	    → 附上落地页链接
	    
	    → 提交发布
	    
	
	5. 在小型产品发布平台提交产品
	    
	    → Uneed .best
	    
	    → Microlaunch .net
	    
	    → Dealhunt .cc
	    
	
	6. 将产品添加到这10个比价平台
	    1. g2 .com
	    2. PriceSpy .co
	    3. getapp .com
	    4. softwareadvice .com
	    5. trustradius .com
	    6. saasgenius .com
	    7. itcentralstation .com
	    8. PriceRunner .com
	    9. saasworthy .com
	    10. Idealo .com
	
	7. 通过 Kickstarter 寻找客户
	    
	    → 访问 kickstarter .com
	    
	    → 找到最近获得融资的创业项目
	    
	    → 在 builtwith .com 查看他们是否在投放广告
	    
	    → 使用 Reply .io 找到决策者邮箱
	    
	    → 直接发送商业合作提案(而不是求职信)

	> [!NOTE] 创业原则
	> 在你产生疑虑的时候——你会产生疑虑——回到“你已经开始在做”这一事实。到现在，你应已经：（1）瞄准了与使命一致、需要解决的某个问题；（2）产生了不依赖他人投资就可以创业的可行想法，这个想法既能解决问题，也能持续经营并让你赚到钱。现在，你需要做的就是继续前进。
