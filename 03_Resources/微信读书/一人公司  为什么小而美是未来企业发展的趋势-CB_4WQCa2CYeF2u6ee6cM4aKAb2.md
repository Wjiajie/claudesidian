### 第2章　以保持小规模为最终目标



###### 高亮

>[!weread4] 企业常常忘记它们当前的支持者——那些已经在倾听、购买，以及参与的人。这些人应该是你企业最重要的人——远比你希望能接触的任何人重要。无论你的观众是十个人、一百个人或甚至一千个人，如果你没有好好对待他们，那么无论你现在在成长或行销方面做任何事情，都不会带来任何改变。请一定要倾听些已经关注你的人，与他们沟通，并且帮助他们。

### 第12章　关系的隐藏价值





###### 高亮

>[!weread4] 成长骇客所建立的关系通常是为了抵销客户流失，它们的目标是尽力的快速建立受众，然后尽可能不断向他们进行销售，直到他们让步、购买，或者放弃然后离开。
 >- 2023-12-26 01:22:39
 >---

>[!weread4] 一人公司透过朝着更好、非更大的方向努力，就能找到它正确的方向，要做到这点，它就要和它的受众与客户建立长期关系
 >- 2023-12-26 01:28:08
 >---

>[!weread4] 当企业透过它们的行动，重复分享简单的讯息，真正的关系就会被建立起来。
 >- 2023-12-26 01:30:54
 >---

>[!weread4] 第三种必备的资本是社会资本（social capital），虽然金融与人力资本很重要，但社会资本往往是企业成败的因素，因为它关系着市场或受众如何看待你的产品的价值。
 >- 2023-12-26 09:33:30
 >---

>[!weread4] 一人公司需要把社会资本想像成银行帐户，你只能从中取出你放进去的东西。
 >- 2023-12-26 09:35:16
 >---

>[!weread4] 相反的，在你要求受众购买你卖的东西之前，你必须经常在你的社会资本帐户中存款，并积累你的余额。你可以尽力多为你的受众提供协助与创造价值，藉此累积社会资本。
 >- 2023-12-26 09:36:43
 >---

>[!weread4] 关于建立社会资本，有几个不同的学派，其中一个受欢迎的理论是由HootSuite公司的山姆．米尔贝斯（Sam Milbrath）所提出，该理论认为先把你跟受众的大量互动分为三等份。山姆建议三分之一的更新应该与你的企业或内容相关，另外三分之一应该是分享他人的内容，最后三分之一应该是与你的受众建立关系的个人化互动。
 >- 2023-12-26 09:41:22
 >---

>[!weread4] 抱持着同理心去了解，除了你的产品或服务——无论是知识、教育、或只是帮助——消费者真正希望从你的一人公司得到的是什么，这样一来能得到很大的帮助。同理心可以让关系从「我能卖给你什么？」变成「我该如何做才能真正帮助你？」这是累积社会资本的方法：建立长期且互利的关系。
 >- 2023-12-26 09:45:31
 >---

>[!weread4] 如果没有优先考虑你的企业所服务的核心群体与关系，你很可能让他们觉得自己不重要——或者更糟的是，让他们觉得你的公司不在乎他们。这时，他们会拾起数位草耙子，带着对你企业的愤怒走上网路世界的街头。而消费者的愤怒，很少会在愤怒的推文上停下来——这也会引起严重的商业不良影响。
 >- 2023-12-26 13:33:37
 >---

>[!weread4] 当你把这件事做好，客户对你的欣赏就会随之而来，一旦你有了他们的欣赏，客户就会对你的成功与成就产生兴趣，而不是产生怨恨或嫉妒的感觉。
 >- 2023-12-26 13:33:37
 >---

>[!weread4] 最后，长期保持关系是很重要的，即使有些客户一阵子没有向你购买东西、提供财务支持，你也应该和他们保持关系。保持一致性与长久性是关键。
 >- 2023-12-26 13:33:38
 >---

>[!weread4] 身为一人公司，你如何让你的客户高兴？
 >- 2023-12-26 13:33:38
 >---

>[!weread4] 如果你在为自己工作，通常可能会更相信自己是孤独的，然后也表现得像你的公司在这场奋斗中是孤独的，好像你的企业只需要你，不需要外部的互动或参与。但是跟同行的其他人、或甚至类似行业的其他人建立连结，并且与他们培养关系，我们有机会获取新想法，也有建立宝贵关系的途径，藉此带来新客户，或者也可以当作情绪宣洩的管道。当然，我们想要保有我们的自主权与独立性，但是我们也时常需要一群人一起运作，因为团结力量大。

### 第13章　创立一人公司——我的故事







###### 高亮

>[!weread4] 如果你不认为你有可能做得更好，或者你不在乎是不是能做得更好，那么做你自己的事情就没有意义了。既然如此，为别人工作也很好——它们已经站稳脚跟，而且有人会处理那些你可能不想做的工作。
 >- 2023-12-26 22:10:13
 >---

>[!weread4] 在我开始向任何人销售任何东西之前，我会以某种方式帮助他们，先跟他们建立关系。
 >- 2023-12-26 22:13:50
 >---

>[!weread4] 这种方法反映了一人公司的精神，你可以马上开始，不需要在资源、工具、或自动化软体上投入大量资金。
 >- 2024-06-05 22:58:38
 >---

>[!weread4] 金钱是许多一人公司从业余专案做起的原因：为了支付创办人的生活费，他们到达MVPr的路程可能需要花一点时间。我刚开始和我父母一起住，抵销我自己的生活费（喂，当时我才十九岁），然后用几年的时间，慢慢从提供服务完全过渡到提供产品——直到产品赚的钱固定会超过我提供服务所收的费用。
 >- 2024-06-05 23:12:48
 >---

>[!weread4] 首先，你要确保在你营业的国家与地区正确设置你的营业人（business entity）。第二，将你的企业独立于你个人之外。换句话说，你的企业本身应该是法人——在大多数国家是公司，或在美国是有限公司（Limited Liability Company，LLC）。这样的话，如果你的企业出问题，是你企业要承担法律责任，而不是你本人。所有的钱都应该直接进入你的企业，而不是直接给你，然后你应该以薪水或分红方式付钱给自己。
 >- 2024-06-05 23:14:13
 >---
