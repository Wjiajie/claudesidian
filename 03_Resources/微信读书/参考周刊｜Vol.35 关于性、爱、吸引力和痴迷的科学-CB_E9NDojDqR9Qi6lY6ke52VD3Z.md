### 关于性、爱、吸引力和痴迷的科学



###### 高亮

>[!weread4] 爱情，不单是情感体验，更是大脑中一系列复杂化学反应的产物。恋爱期间，多巴胺、催产素等神经递质不仅带给我们甜蜜，也触发我们的本能，使我们作出一些非理性的行为。
 >- 2024-01-09 13:20:12
 >---

>[!weread4] 我们将浪漫爱情定义为对另一个人的强烈渴望，并期待它能够延续到未来。
 >- 2024-01-09 13:21:27
 >---

>[!weread4] 另一个显著增加的神经递质是血清素，它在强迫症和抑郁症中也很活跃。这并不是说恋爱会引发抑郁，但它可能会引起类似于抑郁症的内耗状态——即不停地沉迷于对伴侣的思念之中。
 >- 2024-01-09 13:23:11
 >---

>[!weread4] 此外，当你与你的伴侣亲密相处时，你的身体将经历一系列神秘而美好的变化。例如，催产素，这种被称为“拥抱荷尔蒙”的神经递质，在亲密接触时会被释放。
 >- 2024-01-09 13:24:01
 >---

>[!weread4] 这些神经递质的综合作用会使你对伴侣产生强烈的迷恋，甚至上瘾。你会不自觉地想念他们，并渴望时刻与他们紧密相连。实际上，深深爱上某人，在某种程度上是一种甜蜜的负担。
 >- 2024-01-09 13:24:38
 >---

>[!weread4] 同时，我们的前额皮质（一个负责认知功能、较为先进的大脑区域 ）的活跃度开始降低。这个区域通常是我们做决策和进行未来规划的中枢区域。
 >- 2024-01-09 13:25:58
 >---

>[!weread4] 因此，我常建议人们，在决定与某人结婚前，一定要花充分的时间去了解对方。只有当爱情初期的那些强烈情感逐渐平息时，你才能真正清楚地看见你眼前的一切。
 >- 2024-01-09 13:26:30
 >---

>[!weread4] 这些表面看似轻率的一夜情、亲密友谊和同居，实际上可能是人们在婚前对伴侣的一种谨慎选择。过去，婚姻是一段关系的起点，而现在，它似乎成了爱情关系的“高潮”。
 >- 2024-01-09 13:27:19
 >---

>[!weread4] 我曾推测，如果有一个长期相互了解的阶段，那么步入婚姻时，双方已经对彼此有了深刻的认识，知道自己将会建立一个长久、稳定且幸福的婚姻。
 >- 2024-01-09 13:28:03
 >---

>[!weread4] 我认为，这种我所说的“快性慢爱”过程，即通过长时间、仔细地了解对方的方式，有助于我们的大脑作出理性判断。你会了解这个人如何与你的父母相处，他们如何处理友情、金钱、争吵、健康等问题。通过这种深入的了解，我对未来的爱情充满了乐观。
 >- 2024-01-09 13:29:16
 >---

>[!weread4] 你和伴侣在沟通和妥协方面的能力如何？你们是否愿意为了这段关系，做出并非首选的抉择？这是每个人在维系关系中都必须面临和解决的核心问题。
 >- 2024-01-09 13:31:09
 >---

>[!weread4] 研究大脑让我明白，要维持爱情，首先得处理好三个基本的交配和繁殖的大脑系统：性冲动、与伴侣进行性生活、定期与伴侣发生性关系。
 >- 2024-01-09 13:31:50
 >---

>[!weread4] 保持与伴侣的性生活频率，是维系性冲动大脑系统的关键。而为了保持强烈的浪漫情感，尝试一些新鲜事物至关重要。新奇经历能刺激多巴胺系统，让爱情持续新鲜有趣。因此，别忘了，浪漫远不止于卧室的温存。
 >- 2024-01-09 13:32:41
 >---

>[!weread4] 此外，对长久幸福的伴侣进行的研究揭示了大脑的奥秘。这次的研究对象是那些婚姻生活超过二十年但依然深爱着伴侣的人。
   他们的大脑中有三个区域格外活跃：一处与共情相关，一处涉及控制情绪的能力，还有一处与所谓的“积极幻觉”有关。这种能力虽然看似简单，却往往难以做到——就是忽略那些不喜欢的方面，将注意力集中在喜欢的特质上。
 >- 2024-01-09 13:34:39
 >---

>[!weread4] 为了维护一段幸福长久的伴侣关系，大脑建议我们做到以下几点：
   · “选择性忽视”伴侣的缺点，将注意力集中在伴侣的优点和我们所爱的特质上
   · 对伴侣富有同理心
   · 控制好自己的情绪
   · 与伴侣保持良好的性关系
   · 一起尝试新鲜事物
   · 每天说几句赞美对方的话
   你的大脑能助你维持长久而深刻的感情。
   我们为爱而生。
