## 目标
我想构建一个思维模型展示卡片应用，能让用户有兴趣每天学习一些思维模型相关的概念，逐渐提高决策能力。

## 蓝图
应用需要具备以下功能：
1. 主页，展示整个应用的落地页，吸引用户使用和付费。
2. 思维模型详情页面，整个页面展示所有思维模型的标题索引，点击可以跳转到具体内容页面。具体的页面会采用markdown的形式编写，所以网页需要具备markdown解析和渲染的能力。
3. 思维模型书籍推荐页面，采用瀑布流的书籍卡片优雅展示，点击卡片可放大卡片，并提供可以跳转到购买的网站的超链接，以及免费的书籍下载连接。
4. 游戏化学习页面，提供随机的卡片学习打卡，打卡完成后，生成打卡截图，包含打卡天数，思维模型概念以及总结，和该思维模型最相关的名人名言。
5. 思维模型需要按相关度聚类，彼此连接成节点树。
6. 付费购买页面，提供功能介绍以及付费跳转连接。

## 技术栈
基于react + tailwind css + typeScript实现，可以部署到vercel

## 注意
应用能从markdown文件解析思维模型的具体内容，你暂时不需要关注具体思维模型的内容时怎样的，只需要创建一些测试的markdown文件，来测试整个应用的模块是正常的就行。

## SOP
1. 寻找100+个思维模型，完成内容介绍的prompt
	
2. 每个思维模型的文章提供下面的内容
	1. 概念，摘要和分类
	2. 内容介绍，wiki链接
	3. 购买书籍链接，免费下载的链接（看网页如何自动集成书籍下载插件安装）
3. 思维模型卡片
	1. 动画展示，字体，卡片背景选择
	2. 概念，摘要和名人引用

## 推广
基于工作流的视频读书笔记生成，发表到小红书，并提供购买连接。
