# nanobot

> 极简主义的 AI Agent 框架，约 3400 行代码

## 概述

nanobot 是一个极简主义的 AI Agent 框架，专注于即时反馈和高可扩展性。它用极少的代码量实现了完整的 AI 助手功能，是一个"Pythonic"的替代方案。
